# Extract Data from [Zoomit](https://www.zoomit.ir/) site

Extracting data from zoomit website using python programming language. 

## Clone project 
```
$ git clone https://github.com/yasinEnigma/zoomit-data
$ pip3 install -r requirements.txt
$ python3 zoomit.py 

```
